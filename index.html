Быть зарегистрированным на любом из онлайн-ресурсов Maxline и верифицировать свой аккаунт.

Для участия в турнире игроку необходимо сделать выигрышную/-ые ставку/-и с коэффициентом не менее 2.2 в любой игре, организуемой TVBet. Сумма выигрыша по каждой ставке с коэффициентом не менее 2.2 будет записана в турнирной таблице как сумма баллов, где 1 балл эквивалентен 1 EUR на момент расчета ставки.

Пример
Игрок делает первую ставку 5 BYN с коэффициентом 2.5. Сумма выигрыша по ставке 5*2.5 = 12.5 BYN. На момент расчета ставки курс 1 EUR = 2.692 BYN. Соответственно, 12.5/2.692 = 4,64 EUR = 4 балла. Игрок делает вторую ставку 3 BYN с коэффициентом 2.3. Сумма выигрыша по ставке 3*2.3 = 6.9 BYN. На момент расчета ставки курс 1 EUR = 2.712 BYN. Соответственно, 6.9/2.712 = 2,5 EUR = 2 балла.Общее количество баллов 4+2 = 6.
Количество ставок не ограничено. Игрок с наибольшей суммой баллов станет победителем турнира. Если игроки набирают одинаковое количество баллов, то для определения места игрока будет происходит подсчет количества ставок. Высшее место займет тот игрок, который совершил большее количество ставок.
